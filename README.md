# Мобильное приложение GPS-навигатор

## Введение

Мобильное приложение предназначено для построения и визуализации кратчайших маршрутов на территории предприятия ОАО «Северсталь-метиз». 
В функционал приложения входит: 
1. Выбор точки назначения;
2. Построение маршрута до точки назначения;
3. Расчет расстояния до точки назначения;
4. Определение текущего местоположения пользователя. 

Реализованное ПО, после внедрения, уменьшит время погрузки грузовых автомобилей и улучшит ориентирование по территории завода для новых сотрудников и водителей грузового автотранспорта, также повысит уровень удовлетворенности клиентов (CSI) и увеличит индекс потребительской лояльности (NPS). Данное руководство пользователя содержит информацию для пользователя об эксплуатации приложения.

## Выбор точки назначения

Для того, чтобы построить маршрут до точки назначения необходимо открыть всплывающее меню, для этого необходимо нажать на поле с надписью «Ввести точку назначения…» (рис. 3).

</p>
<p align="center">Рис. 3. Вызов выпадающего списка<p align="center">

В данном всплывающем меню выводится список всех точек назначения (рис. 4). В правом углу меню находится скроллбар (ползунок) изменяющий своё положение в зависимости от вращения списка. Скроллбар также отображает изменяет свой размер в зависимости от количества элементов в списке: чем меньше точек назначения, тем шире ползунок и наоборот.

## Построение маршрута

Для построения маршрута до точки назначения необходимо только нажать на наименование точки назначения. После выбора будет построен маршрут (рис. 7). Маршрут подсвечивается сине-красной линией. После выбора точки назначения появляется информационная панель, на которой отображена информация о точке отправления и точке назначения, также на панели имеется кнопка отмены построения маршрута.

## Управление камерой
